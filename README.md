# Erasmus Curriculum Database Integration

This project designs a relational database system for integrating and comparing 
Erasmus exchange program curricula between different universities.

The goal is to enable students and administrators to identify course equivalencies 
across different Computer/Informatics departments.

## Universities Analyzed

- Izmir University of Economics – Software Engineering
- Istanbul Technical University – Computer Engineering
- Dokuz Eylul University – Computer Engineering

## Project Structure

analysis/ -> curriculum data analysis  
eer_diagrams/ -> conceptual EER models  
relational_model/ -> relational schema design  
sql/ -> SQL scripts for implementation  

## Features

- Curriculum comparison
- Course equivalency mapping
- ECTS credit alignment
- Prerequisite tracking
- Data integrity enforcement using constraints and triggers
- Relational schema design based on EER model

## Technologies

- SQL
- PostgreSQL 
- EER Modeling
- Relational Database Design
